# Зогсоолын торгуулийн систем

Энэ төсөл нь `Vite + React + TypeScript + Supabase` дээр суурилсан MVP систем.

## Үндсэн боломж

- 3 хэрэглэгчийн эрх: `Супер админ`, `Ажилтан`, `Хэрэглэгч`
- Торгуулийн суурь дүн:
  - Суудлын машин: `40,000 ₮`
  - Жийп: `50,000 ₮`
  - Ачааны машин: `60,000 ₮`
  - Автобус: `80,000 ₮`
- Хоногийн төлбөр: `6,000 ₮`
- Нийт дүн: `суурь торгууль + (хоног * 6,000 ₮)`
- Хэрэглэгч дугаараараа шалгаж төлбөр төлнө
- Ажилтан төлбөрийн жагсаалт дээр төлбөр баталгаажуулж, бэлтгээд гаргана

## Локал орчинд ажиллуулах

1. Хамаарлууд суулгах:
   - `npm install`
2. `.env.local` файлд тохиргоо оруулах:
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`
3. Supabase дээр `supabase-schema.sql` файлыг бүтнээр нь ажиллуулах
   - Storage bucket: `impound-images`
   - Tables: `parking_cases`, `payments`, `parking_case_images`, `audit_logs`
4. Төслийг асаах:
   - `npm run dev`
